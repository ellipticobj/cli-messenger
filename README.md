# cli-messenger
simple cli messenger app made with python

# running
git clone this repo
`git clone https://github.com/ellipticobj/cli-messenger.git`

## server 
to run the server, run server.py 

```bash
python server.py
```

you can use something like tmux to keep it running

## clients
run client.py
